Анонимный бот для постов в группе Telegram (может выкладывать в определенную тему если выключены)

Инструкция по использованию:

1) Скачайте и распакуйте файл
2) Скачайте и установите Python (поддерживаемая версия 3.10.11)
3) Откройте командную строку и зайдите в папку bot с помощью "cd"
4) Установите зависимости - pip install -r requirements.txt
5) Откройте @BotFather, пропишите /newbot, введите имя, и свободный id. Скопируйте ip и вставьте в .env [BOT_TOKEN=]
6) Придумайте пароль и запишите в .env [BOT_PASSWORD=]
7) Добавьте созданного бота в нужную группу
8) Назначьте бота администратором группы
9) Пропишите в чате / любой теме : /bind_group
10) Зайдите в диалог с ботом и пропишите : /start
11) Введите пароль и выбирайте нужное действие
12) Для настройки ролей перейдите в roles.txt и введите там по примеру: Роль - @username, @username или же : Роль - @username
