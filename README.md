Анонимный бот для постов в группе Telegram (может выкладывать в определенную тему если выключены)

Инструкция по использованию:

Скачайте и распакуйте файл
Скачайте и установите Python (поддерживаемая версия 3.10.11)
Откройте командную строку и зайдите в папку bot с помощью "cd"
Установите зависимости - pip install -r requirements.txt
Откройте @BotFather, пропишите /newbot, введите имя, и свободный id. Скопируйте ip и вставьте в .env [BOT_TOKEN=]
Придумайте пароль и запишите в .env [BOT_PASSWORD=]
Добавьте созданного бота в нужную группу
Назначьте бота администратором группы
Пропишите в чате / любой теме : /bind_group
Зайдите в диалог с ботом и пропишите : /start
Введите пароль и выбирайте нужное действие
Для настройки ролей перейдите в roles.txt и введите там по примеру: Роль - @username, @username или же : Роль - @username
